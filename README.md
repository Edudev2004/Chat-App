# Chat App
Aplicación de chat en tiempo real para aprender desarrollo fullstack.

## Tecnologías
- **Backend:** Node.js, Express, Socket.io, MongoDB
- **Frontend:** React, Bootstrap, Socket.io-client

## Características
- Chat en tiempo real
- Salas de chat
- Usuarios online/offline
- Historial en MongoDB
- Interfaz bonita
